# DOGPICS

Welcome to DOGPICS, an app that displays pictures of dogs.  

## Technologies Used

- Python (3)
- Flask
- Imgur
- Google Cloud Platform (for hosting)
- Github

WIP: Image Recognition, to know if picture is a dog or not
